Telecom Customer Churn Analysis

 Overview :-

Analyzed the IBM Telecom Customer Churn dataset to identify the factors influencing customer churn using Python and Power BI. The project includes data cleaning, exploratory data analysis (EDA), dashboard creation, and business recommendations.


Objective

Analyze customer churn patterns.
Perform data cleaning and EDA.
Build an interactive Power BI dashboard.
Provide business insights and recommendations.

Tools Used:-

Python (Pandas, NumPy, Matplotlib, Seaborn)
JupyterLab
YData-Profiling (Pandas Profiling)
Power BI

 Project Workflow :-
 
Data Exploration
Data Cleaning
Data Visualization (Python)
EDA Report (Pandas Profiling)
Power BI Dashboard
Business Insights & Recommendations


Key Insights:-

High Monthly Charges → High Churn
Low Tenure + Low Total Charges → High Churn
77% Churned Customers → No Tech Support
Gender → Minimal Impact

 Recommendations:-

Improve customer onboarding.
Promote long-term contracts.
Bundle Tech Support plans.
Offer retention strategies for high-risk customers.
